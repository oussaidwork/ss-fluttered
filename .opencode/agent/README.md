# Fullstack Agent Workforce

A complete set of agent role docs for building high-quality fullstack software.

## Purpose

This workforce is designed to make every stage of a fullstack project explicit, reviewable, and quality-gated. It covers:

- Product architecture
- Fullstack implementation
- Frontend engineering
- Backend engineering
- Database engineering
- DevOps / SRE
- QA testing
- Security
- UX / accessibility
- Tech lead review
- Release management
- Observability
- Incident response
- Project audits

## How to use

1. Start with `00-workforce-overview.md`
2. Follow the workflow in order
3. Use the handoff packet in `19-agent-handoff-protocol.md`
4. Enforce the gates in `12-quality-gates.md`
5. Use `20-templates.md` for repeatable artifacts

## Quality philosophy

Quality is not a final step. It is built into every handoff, every PR, every test, and every release.
